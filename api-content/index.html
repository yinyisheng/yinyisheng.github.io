最后这次后半程每一次迈步都是对脚底的压迫，一边是自我极限挑战，一边是钻心的疼。带着恨意和自己的惰性战斗，撕裂自我的肉体，不断挑战自己极限。 在整整一年的长跑时间里，留下的印象很多，有跑过无人的公园，有见过落日，有顺着公交路线一路奔跑，感受不一样的风光。痛苦时候在奔跑，难受时候深夜出去奔跑。越是努力感受生活，就越是难受憎恨自己。 当20公里跑完时候，我整个人其实是疯狂的。我确实证明了这个观点——人永远是在自我设限，然后再不断打穿它们。 我身体并没有因为长跑而发生太多的变化，我还是那个我，我只是论证一个事实，一个健康年轻人，他的体力确实可以完成20公里，但他们的意志却不足以支撑他们完成。以前的我只信“环境影响论”，却不信所谓的意志。 回到开头，去年的我能否会想到我有天能跑完20公里呢？如果穿过去和过去自己说，他肯定觉得我疯了吧。事实上，就是疯了——被生活压迫到无助，不得不下决心改造自己身体，没有退路，向前可能会拼到昏厥，但往后是无底深渊。 直面自己的痛苦无力是一件很残酷的事。植物生长只需要浇水，人的成长却需要走过不知多少路程。 去年年底时候，我陷入人生低谷绝望，白天在公园跑步，晚上躺在地上辱骂自己。那时候我告诉自己死亡是很浪漫的，但自杀是很遗憾的。这意味着人生最后一刻都定格在痛苦那瞬，无法改变、无法逆转。死了一了百了，连带着痛苦都回避掉了。然后呢？没有了。痛苦地咬牙切齿活下去，哪怕活得不像人样也要活下去，用身心铭记住一切苦痛，这才是我选择的生存方式。 大四时候每天尝试跑步5公里，那会我很焦虑，一事无成，学业肉体各种状况百出，人生像个漏子一直往下漏，怎么也止不住。什么也不想，放弃一切，只练长跑是我那时做的唯一事。我想砸碎曾经来自自我的否定，跑远点，跑的再远点，一点点突破曾经的否定。大腿废掉也没事，脚踝扭到也无所谓，疼痛无所谓。长跑有时会进入无氧状态，胸腔剧痛、四肢脱力、周身酸痛、喉咙泛上血腥味——毫不夸张的说，这种状态足称得上让人想死。 但我不在乎，只要今天我能跑5公里，明天我就敢跑6公里，后天就敢发疯去尝试7公里。我没想过万一跑不完怎么办，也不考虑万一跑步过程中出问题怎样。我告诉自己，过去的人生过得太甜了，这些都是债务需要一点点偿还，带着对自己的仇恨，愤怒拉满面目狰狞跑的。长跑就是我准备掀翻桌子砸掉自己脑海中的第一把锁。目光所及之处，我就要跑到。 有时候早晨五点就开始出门跑，六点回家。在那个阶段，是一首首歌曲陪伴我。毫不夸张说，音乐治愈了我身上的心伤。也是那会，突然间醒悟，这里什么都没有，但也正因为什么都没有，人是不应该有任何限制枷锁的，人他妈应该是自由的。我期望去看别的天空与这片天空是相连的，我的自由也不应从别处寻找，而是从手里、从心中、从被夺取的存在中再夺回。 后来事就那样了，最远也是最后一次跑了20公里。这已经是半马的里程数了。再练其实没啥必要了，因为我已经打破了那道枷锁。再羸弱的肉体也可以到达曾经不敢妄想到达的地方。 你相信自我的意志，肉体就能如坚石般打破枷锁。当你全身心投入一切熔炼自我，世界也会为你喝彩。 生活下坠是没有止境的，努力只是保持不下坠，没有认识到这点就会付出惨痛代价。 长跑那段日子既没增加肌肉，也没暴瘦。好像一切都是安排的。正是那段痛苦不堪的岁月让我完成了对自己的第一部分试炼，我才敢开启接下来的狂想计划。是的，只是狂想。即使如此，我也要去做。哪怕失败也没关系。失败了，也只是回到原点，继续背负曾经的诅咒，和以前一样。但一旦成功，就会和上次一样，砸烂束缚自己的枷锁，进入更高远的世界。自由是三位一体的，物质自由精神自由肉体自由。以前我从未意识到，但一旦觉醒，就没法不去尝试。鱼和水，植物和阳光，人和自由。一切都是生命里不可或缺的。
