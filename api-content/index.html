有时候早晨五点就开始出门跑，六点回家。在那个阶段，是一首首歌曲陪伴我。毫不夸张说，音乐治愈了我身上的心伤。也是那会，突然间醒悟，这里什么都没有，但也正因为什么都没有，人是不应该有任何限制枷锁的，人他妈应该是自由的。我期望去看别的天空与这片天空是相连的，我的自由也不应从别处寻找，而是从手里、从心中、从被夺取的存在中再夺回。 后来事就那样了，最远也是最后一次跑了20公里。这已经是半马的里程数了。再练其实没啥必要了，因为我已经打破了那道枷锁。再羸弱的肉体也可以到达曾经不敢妄想到达的地方。 你相信自我的意志，肉体就能如坚石般打破枷锁。当你全身心投入一切熔炼自我，世界也会为你喝彩。 生活下坠是没有止境的，努力只是保持不下坠，没有认识到这点就会付出惨痛代价。 长跑那段日子既没增加肌肉，也没暴瘦。好像一切都是安排的。正是那段痛苦不堪的岁月让我完成了对自己的第一部分试炼，我才敢开启接下来的狂想计划。是的，只是狂想。即使如此，我也要去做。哪怕失败也没关系。失败了，也只是回到原点，继续背负曾经的诅咒，和以前一样。但一旦成功，就会和上次一样，砸烂束缚自己的枷锁，进入更高远的世界。自由是三位一体的，物质自由精神自由肉体自由。以前我从未意识到，但一旦觉醒，就没法不去尝试。鱼和水，植物和阳光，人和自由。一切都是生命里不可或缺的。
